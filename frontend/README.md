# frontend

In the parent directory, make sure to run `yarn install`, `yarn compile`, then `yarn postcompile`.

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```

### Lints and fixes files
```
yarn lint
```
